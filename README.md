### In menus:

|  Key  |         Action         |
|:-----:|:----------------------:|
| Enter | Choose Selected Action |
|   ↑   |    Scroll Actions Up   |
|   ↓   |   Scroll Actions Down  |
|   Z   |   Surrender            |
